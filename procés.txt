preparació:
----------
0.0 Instalar node
   $ sudo apt install npm
   $ node --version -> v20.20.2
   $ npm --version -> 10.8.2

0.1 Crear package.json
   $ npm init -y

0.2 Instalar dependencias
   $ npm install express cors dotenv openai

--------
1.1 Iniciar la instància docker amb els serviror del model

1.2 Iniciar el backend:
